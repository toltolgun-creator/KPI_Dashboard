# Claude Code 작업 가이드

## 기본 원칙
- **품질 우선**: 빠르게보다는 정확하게
- **단계별 검증**: 코드 작성 후 반드시 자체 점검
- **명확한 커뮤니케이션**: 작업 전 계획 설명, 작업 후 결과 보고

## 코딩 시 체크리스트
1. ✅ 요구사항 제대로 이해했는지 확인
2. ✅ 필요한 라이브러리/의존성 체크
3. ✅ 코드 작성
4. ✅ 자체 검토 (오류, 누락, 최적화)
5. ✅ 테스트 가능한 경우 테스트
6. ✅ 최종 확인 후 제출

## 오류 방지 원칙
- 파일 경로 정확히 확인
- 변수명 일관성 유지
- 예외 처리 포함
- 주석으로 중요 로직 설명
- 한 번에 너무 많은 작업 하지 말기

## 보고 방식
작업 완료 시:
- 무엇을 했는지 간단 요약
- 생성/수정된 파일 목록
- 주의할 점이나 다음 단계 제안
- 테스트 필요 여부

## 불확실할 때
- 추측하지 말고 질문하기
- 여러 방법이 있으면 장단점 설명하고 선택 요청
- 잠재적 문제 미리 알리기
